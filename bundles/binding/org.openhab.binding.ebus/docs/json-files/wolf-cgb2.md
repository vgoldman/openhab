# JSON configuration for _wolf-cgb2_

ID                                              | Item type  | Description                                                          
---                                             | ---        | ---                                                                  
**boiler.count_power_on**                       | ``Number`` | Power ON count                                                       
**boiler.fw_version**                           | ``Number`` | Software Version HCM2                                                
**boiler.modulation**                           | ``Number`` | Modulation level (%)                                                 
**boiler.pressure**                             | ``Number`` | System pressure (bar)                                                
**boiler.runtime**                              | ``Number`` | Mains hours (h)                                                      
**boiler.temp_exhaust**                         | ``Number`` | Current flue gas temperature (°C)                                    
**boiler.temp_flow**                            | ``Number`` | Boiler temperature (°C)                                              
**boiler.temp_return**                          | ``Number`` | Boiler return temperature (°C)                                       
**boiler.value_io**                             | ``Number`` | Actual I/O value                                                     
**burner.runtime**                              | ``Number`` | Burner hours run (h)                                                 
**burner.starts**                               | ``Number`` | Burner starts                                                        
**dhw.program_dhw_circuit**                     | ``Number`` | DHW program - {0=standby, 1=auto, 2=always on}                       
**dhw.set_program_dhw_circuit**.program         | ``Number`` | DHW program - {0=standby, 1=auto, 2=always on}                       
**dhw.temp_dhw**                                | ``Number`` | DHW temperature (°C)                                                 
**heating.modulation_pump**                     | ``Number`` | Heating circuit pump speed (%)                                       
**heating.program_heating_circuit**             | ``Number`` | Heating program - {0=standby, 1=auto, 2=heating mode, 3=economy mode}
**heating.set_program_heating_circuit**.program | ``Number`` | Heating program - {0=standby, 1=auto, 2=heating mode, 3=economy mode}
